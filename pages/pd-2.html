Disney+ Clone

Project information

  • Type: Front-End
  • Technologies:   VUE.JS | SASS | HTML | JAVASCRIPT
  • GitHub Repository

Questo progetto è un clone di Disney+ sviluppato con Vue.js e Axios, che integra dati dinamici dalle API di TheMovieDB per popolare categorie come Disney e Marvel. Il progetto sottolinea l'uso di componenti per un'interfaccia responsiva, dimostrando competenze in front-end development e design adattivo. La struttura modulare e l'esperienza utente fluida evidenziano la mia capacità di creare applicazioni web moderne e reattive. Questa esperienza ha rafforzato le mie abilità in Vue.js, Axios e nel garantire un'esperienza utente coesa su dispositivi diversi, preparandomi per future sfide nello sviluppo web.

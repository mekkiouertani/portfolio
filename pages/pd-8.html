BDoctors

Project information

  • Type: Full-Stack
  • Technologies:   LARAVEL | VUE.JS | SASS | HTML | MySQL
  • Fron-end GitHub Repository
  • Back-end GitHub Repository

Sviluppato in team di un sito web responsivo per medici utilizzando Laravel e MySQL per il back-end e Vue.js con Bootstrap per il front-end. Il progetto include una dashboard che consente ai medici di registrarsi, gestire profili e acquistare pacchetti di sponsorizzazione, aumentando così la loro visibilità nei risultati di ricerca. I visitatori possono cercare i medici per specializzazione, valutazioni e numero di recensioni, accedendo a dettagli completi sui professionisti, gli utenti possono lasciare recensioni e contattare i medici via email tramite un form con un sistema di validazione a 2 livelli.

To do List (Death Note)

Project information

  • Type: Front-End
  • Technologies:   VUE.JS | CSS | HTML | JAVASCRIPT
  • GitHub Repository

Una to-do list ispirata a Death Note, realizzata con Vue.js. Questa applicazione unisce funzionalità di gestione attività a un'estetica unica, con una copertina che, al doppio click, si apre rivelando le note all'interno. L'interfaccia, che richiama il celebre quaderno del manga, offre un'esperienza d'uso coinvolgente e originale. Implementando Vue.js, il progetto dimostra come personalizzare le applicazioni standard con stili creativi, mantenendo allo stesso tempo un codice pulito e modulare. Questa to-do list non è solo uno strumento pratico ma anche un omaggio a Death Note, evidenziando le mie capacità di innovare nell'UX/UI design attraverso l'uso di tecnologie front-end moderne.
